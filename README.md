- 01 品牌核心

- 04 Color 色彩系統(點擊複製 HEX、品類色即時切換)
- 05 Typography 字體系統(即時字級預覽)
- 06 Brand Voice 品牌語氣(推薦 / 禁用字對照、文案弱 / 強切換)
- 07 Photography 攝影風格
- 08 Graphics / Icons 圖形與圖示(Lucide 12 icon 展示)
- 09 Charts 資料視覺化
- 10 Brand Applications 品牌應用
- 11 商標版權與高風險詞庫(關鍵字即時搜尋)
- 附錄 待補資料與暫定規範
